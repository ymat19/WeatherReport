# 傘判定 Web アプリ

## 概要

気象庁の API(非公式)を用いて、その日に雨が降るかどうかチェックします。雨じゃなければ時計が表示されます。
スマートディスプレイで開く想定です。

## 公開ページ

https://ymat19.github.io/WeatherReport/

## 注意事項

- このアプリは、気象庁の API を非公式に利用しています。API の変更やアクセス制限などにより、予期せぬエラーやデータの取得不可が発生する可能性があります。
